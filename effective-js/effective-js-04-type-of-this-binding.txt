انواع دیس باندیدگ در جاواسکریپت

Implicit this biding
این مورد دیفالت هست و هر وقت از کیوورد فاننکشن استفاده میکنیم انجام میشه
در واقع هر فانکشی که تبدیل به متد بشه دیس در اون فانکشن بایند میشه با ابجکت صدا زننده

Explicit this biding or manual this biding
در مواردی میتونیم خودمون دستی دیس رو بایند کنیم
یعنی به ی فانکشن بگیم بیا از این دیس استفاده کن
function(){ ... ].bind( this )

Forward this biding
در مواردی که یه فانکشی رو پاس میدیم اگر دیس در این فانکشن باید نشده باشه
در هنگام اجرا در ی جای دیگه دیس به درستی بایند نمیشه مثلا در اجرای
Array.map
میتونیم دیس رو به مپ معرفی کنیم
اگه از ارو فانکشن استفاده کنیم نیازی به این مورد نداریم
arrow function () => { ... }

Super class this binding
موقعی که یه کلاس فرزند بخواد از کلاس پدر خودش چیزی رو به ارث ببره
در نسخه ۵ جاواسکریپت از کال مثلا استفاده میکنیم
Parent.call( this, x, y )
میگه کلاس پرنت روی دیس من هم اکس و وای رو اضافه کن
اما در نسخه ۶ کیوورد سوپر جایگزین شده
super( x, y )
یا در ریاکت
super( props )

Fixed this biding
در مواردی که نمیخوایم
Implicit this biding
اتفاق بیافته چون میدونیم اگه اتفاق بیافته به
Error
یا
Exception
میخوریم میام دیس را به صورت دایم باید میکنیم
مطمنم توی ریاکت این رو بارها انجام دادین
this.onClick = this.onClick.bind( this )
این عبارت یعنی چی؟
یعنی فانکشن آن کلیک هرجا رفتی و اجرا شدی دیس خودت رو گم نکن
به همین سادگی
