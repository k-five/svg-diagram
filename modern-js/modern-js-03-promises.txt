چند نوع فانکش در جاواسکریپت داریم؟ خب خیلی
در جاواسکریپت نمونه از یه کلاص رو با چی میسازیم؟
function constructor

یه پرامیس یه کلاس یا یه فانکشن کانستراکتور هست
چه پارامتری میگره؟ فقط یکی یه کال بک میگریه
new Promise( callback );

خود این کال بک دو تا کال بک دیگه میگره
function callback( fn1, fn2 ){ ... }

چه کسی این دوتا کال بک رو صدا میزنه؟ مرورگر
کی صدا میزنه؟ هر وقت به نتیجه برسن
روی چه نخی؟ روی نخ اصلی نیست

کدوم رو صدا میزنه؟
بستگی به این داره که خطای رخ بده یا نه

اگه خطا رخ نداد؟
اگه نداد کال بک اول
در غیر این صورت کال بک دوم

یعنی ترتیب اجرا مهم نیست؟
چرا هست اگه کال بک دوم رو قبل اول صدا بزنیم
دیگه  کال بک اول اجرا نمیشه

یعنی چی؟
یعنی هر دو حالت باهم اتفاق نمی افتن
یا میشه یا نمیشه
یا موفق یا ناموفق

از کجا اومده این داستان
functional programming

اجرا کننده یکی دیگه س
ولی کنترل کننده ما هستیم

در
high order function
هم داریم
اجرا کننده یکی دیگه ی
ولی کنترل کننده ما هستیم
